# Project Memory

## Core
Sports documentation platform "SportDoc". Primary green #258B58 / #1D9E75. DM Sans body, Plus Jakarta Sans headings.
Lovable Cloud backend. 30 sports with sport-specific terminology.
Coach auth (email + Google). Parent portal via shareable token links (no login).
Multi-academy support with RLS via academy_members.
Razorpay billing: Free (₹0), Club (₹799/mo), Pro Analyst (₹1,999/mo).
